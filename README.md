# KClient

### Descripción

Este proyecto saca un promedio y su desviación estándar en base de todas las edades de los clientes.

### Tecnologías Usadas
* Java 8
* Spring Boot
* Swagger
* Mongodb
* Lombok
* Log4J2

### Consumir Swagger
* dev -> http://localhost:9000/swagger-ui.html
* prod -> http://localhost:9000/kclient/swagger-ui.html
